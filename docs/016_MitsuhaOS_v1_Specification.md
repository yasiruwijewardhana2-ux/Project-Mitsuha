# Overview

MitsuhaOS is the operating system of Project Mitsuha.

Its responsibility is to control every subsystem of the robot and make Mitsuha behave like a living companion instead of a simple machine.

MitsuhaOS separates hardware control, AI processing, memory, behavior, emotions and cloud communication into independent modules.

Every response must include personality.

Every movement must have purpose.

Silence is also communication.

The robot should always appear alive.
