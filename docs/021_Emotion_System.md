
# Mitsuha Emotion System

Version: 1.0 (Draft)

---

# Overview

The Emotion System controls how Mitsuha expresses feelings.

Emotions influence

- Eye animations
- Voice tone
- Body movement
- Head movement
- Idle behaviors
- Conversation style

Mitsuha does not fake emotions.

It selects the most appropriate emotional state based on context.

# Emotion Memory

Previous emotions influence future behavior.

Example

If the owner has been studying for several hours,

Mitsuha becomes

Supportive

↓

Calm

↓

Motivating

instead of becoming playful.

---

# Design Rule

Emotion should always feel natural.

Never random.

Never exaggerated.

Every emotional change must have a reason.
