# Project Mitsuha

## Vision

Project Mitsuha is an AI companion robot inspired by Anki Vector but designed to become more intelligent, more emotional, and more useful.

The goal is not to build only a robot.

The goal is to create a living AI companion.

Mitsuha should:

- Recognize people
- Remember conversations
- Learn from experience
- Express emotions
- Understand the owner's habits
- Move autonomously
- Interact with smart home devices
- Continuously improve through cloud updates

MitsuhaOS will power every part of the robot.

This project is completely custom built.
